# Unity 2D Survival RPG: 7-Days-of-Cultivation
A survival game with weekly cycles, simple leveling to defeat ambushing monsters.

## Overview
A small single-player 2D survival RPG built with Unity.
The game focuses on character training, stamina management,
and weekly monster invasions.

## Features
- Day-based time system
- Character training & stats
- Stamina & HP survival mechanics
- Fishing and economy system
- Weekly monster combat

## Technologies
- Unity (2D)
- C#
- GitHub for version control

## Documentation
- BRD: Business Requirement Document
- SRS: Software Requirement Specification
- Use Case Diagrams

## Purpose
This project was created as a personal learning project
to demonstrate Unity fundamentals and software documentation
skills for internship applications.
